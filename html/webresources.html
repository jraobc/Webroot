Web Resources

W3 Schools

Well organized and easy to understand Web building tutorials, built by the World Wide Web Consortium. Includes HTML, CSS, JavaScript, SQL, Python, PHP, Bootstrap, and a ton more. Really useful for burgeoning web designers, as well as engineers who aren't as familiar with the design side of things.

JS Bin

A live pastebin for HTML, CSS & JavaScript. Good for rapid prototyping of websites and features. Most useful for designers and frontend programmers

DevDocs

A searchable interface for API documentation. Contains docs for basically every web technology a designer or engineer could need.

w3af

A vulnerability scanner and penetration testing toolkit for web apps. Great for full stack or back end engineers, especially those on smaller projects without dedicated security support.

MDN Web Docs

A collection of web development tutorials from Mozilla. Great for getting beginners comfortable with multiple technologies.
